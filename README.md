
# TODO List App

A simple TO-DO application that use the `localStorage`


## Features

- Responsive Design
- Cross platform


## Tech Stack

**Client:** React, TailwindCSS



## Run Locally

Clone the project

```bash
  git clone https://github.com/ertugrulhaskan/todo-list.git
```

Go to the project directory

```bash
  cd todo-list
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run serve
```
